# 📦 StockPro Ultra v2.0

Sistema inteligente de gestão de estoque e margem de lucro para revendedores.

## 🚀 Funcionalidades
* **Controle de Estoque:** Adicione e remova unidades com um clique.
* **Cálculo de Lucro:** Veja quanto você ganha em cada unidade e no total.
* **Busca Inteligente:** Encontre produtos rapidamente pelo nome.
* **Modo Escuro:** Design moderno e focado na usabilidade.
* **Persistência de Dados:** Os dados ficam salvos no seu navegador (LocalStorage).

## 🛠️ Como usar
1. Abra o arquivo `index.html` em qualquer navegador.
2. Cadastre seus produtos com preço de custo e venda.
3. Use os botões `+` e `-` para atualizar o que você tem em mãos.
4. Clique em **Exportar** para salvar um backup dos seus dados em JSON.

## 📱 Dica Mobile
Para usar como um aplicativo no celular:
1. Abra o link do site no Chrome ou Safari.
2. Vá em "Compartilhar" ou "Opções".
3. Clique em **"Adicionar à Tela de Início"**.
